    Description

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value

    mandatory, can be: gmobile, g540, g720, g960, g1200, g1440

    		Stringnone

    Content

    This mixin allows extra content to be passed (through the @content directive).

    Example

        width: auto;
    }

    ibm-type

    @mixin ibm-type($fontVariant: $defaultFont) { ... }

    Description

    Common central mixin used to prefix any style with "ibm-type". You can pass a basic font-family name, or you can pass any attributes through @content

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $fontVariant

    optional

    		String or Variable$defaultFont

    Content

    This mixin allows extra content to be passed (through the @content directive).

    Example

    simple font variant for special mono font to be used when  .ibm-type class exists
  @include ibm-type($defaultMonoFont);
    whole code block style to be encapsulated/prefixed with .ibm-type including default font variant
  @include ibm-type(){
   font-size: 1.25rem;
  }
