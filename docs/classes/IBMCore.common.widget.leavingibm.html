IBMCore.common.widget.leavingibm Class

Defined in: src/js/sources/ibm/common/widgets/leavingibm.js:1
Module: IBMCore

Leaving IBM widget.

This is an overlay nofication that tells the user that the service they are about to engage with is being performed on behalf of IBM, but on a non-IBM web site.

NOTE: DO NOT put this on all external links. Read above for the exact specific context where this is needed.

Requirements
The link's @href must be prefixed with "http://www.ibm.com/links/?url=".
The link's URL (url param) must be URL encoded if it contains any parameters.

Usage:

<a data-widget="leaving" class="ibm-external-link" href="http://www.ibm.com/links/?url=http://www.lanyrd.com">Register on Lanyrd</a>


Usage with URL with params:

<a data-widget="leaving" class="ibm-external-link" href="http://www.ibm.com/links/?url=http://www.lanyrd.com%3Fsomeparam%3Dvalue%26xyz%3Dasdf">Register on Lanyrd</a>

__Event usage__: Subscribe to the IBM widget init event that fires on your DOM element to know when the widget has been initialized.
__Special case:__ In scenarios where you dynamically inject HTML post page load, you can initialize the widget on your injected element like this: 
jQuery("<your_link_selector>").leavingibm();

To destroy the widget and return back the original HTML that the widget initialized on:

jQuery("<your_link_selector>").data("widget").destroy();
