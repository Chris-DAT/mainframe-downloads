IBMCore.common.widget.videoplayer.kaltura Class

Defined in: src/js/sources/ibm/common/widgets/videoplayer-kaltura.js:1
Module: IBMCore

Kaltura video widget.
Dependency: videoplayer.js
View the videoplayer.js for all implementation and options detailer. It is the main controller class that creates the proper object and setup.
This is the video player sub-class for the Kaltura embedded video player object.

Event usage: Subscribe to the IBM widget init event that fires on your DOM element to know when the widget has been initialized and you can access its APIs.
