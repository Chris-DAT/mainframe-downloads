IBMCore.common.util.coreservices Class

Defined in: src/js/sources/ibm/common/util/coreservices.js:4
Module: IBMCore

NOTE: No longer in service. Shutdown mid-2018
Provides the functionality to call the SWWIF services/bundled service call.

Doesn't have to be bundled, can be called single request instantly, but bundled saves # of requests.
It's like a bus leaving the station... Give everyone time to buy a ticket and get on the bus. At a certain time the bus is leaving the station with whomever is on it. However, if you miss the bus you can always just hail a taxi for yourself whenever you want ("makeRequest" function).

Function

Defined in src/js/sources/ibm/common/util/coreservices.js:62

Allows modules to add a request to the bundled core services (SWWIF) call as they are read in the JS.
Execution of the bundle doesn't happen until translations + DOM ready event is published.
This ensures meta & translations are loaded, and waiting for DOM ensures any other shit is loaded and ready.

Parameters:

  • serviceId Integer

    The service's ID/#.

  • callback Function

    The function to call on service response.

  • params Object

    Data/Parameters to add to the service call.

  • secureUrl Boolean

    If this request should use the secured URL (behind IBMidaaS) or not. Only use if you KNOW the user is signed in. The request will fail if user is not already authenticated.

Returns:

  • Function

    The callback function.
