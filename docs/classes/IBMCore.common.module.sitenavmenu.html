IBMCore.common.module.sitenavmenu Class

Defined in: src/js/sources/ibm/common/modules/sitenav-menu.js:1
Module: IBMCore

Site navigation menu

This is the site navigation menu bar that's above the leadspace. It's got the site name on the left and links and drop-down menus on the right.
By default the menunav is sticky. Stickyness can be disabled by adding the attribute to the div: data-sticky="false"

init

()

Defined in src/js/sources/ibm/common/modules/sitenav-menu.js:44

Only Called from DOM ready IF A MENU NAV ELEMENT EXISTS.
If you are injecting it dynamically, call this after you inject it.
Called via setTimeout since it's non-essential and just sets up minimal interaction and adjustments.
This sets up the menunav for stickyness, adjusts drop-down menus for right-alignment, and dynamically show/hides the links based on the browser width and if they can all fit or not.
