IBMCore.common.module.localpagenotification Class

Defined in: src/js/sources/ibm/common/modules/localized-page-notification.js:1
Module: IBMCore

PILOT started April 20 2017:
If a user is detected in a country that's different than what the page is tagged for, and there is a (EN)- custom page available (coded via alt-lang tag)... then show a notification bar at the top of the page to ask the user if they want to goto that localized page.
"Go to" means that we take the href from the alt-lang tag and set it as the href of the button in the notification bar.

Steps how it works:
1. Need to know what country the user is in.
2. If so: Is the page they are on a different country code?
3. If so: Is there a link tag with a locale with "en-<user_CC>"? (pilot is for EN enabled CCs only because no xlation needed)
4. Wait for locale selector ready to grab the text value.
5. Show notification bar.
