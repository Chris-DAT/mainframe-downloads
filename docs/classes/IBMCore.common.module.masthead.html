IBMCore.common.module.masthead Class

Defined in: src/js/sources/ibm/common/modules/masthead.js:17
Module: IBMCore

Masthead.

This is the main file for the masthead and everything associated with it.
This waits for translations to be available and onload will create the main masthead links and associated functionality.

Available options/settings: "alternate": Same as the default, but WITHOUT the inline main masthead links. "popup": For use ONLY on popup pages. Only the logo shows in the masthead.
