IBMCore.common.widget.videoplayer.ustream Class

Defined in: src/js/sources/ibm/common/widgets/videoplayer-ustream.js:1
Module: IBMCore

Ustream video widget.
Dependency: videoplayer.js
The videoplayer.js is the main controller that creates the proper object and setup.
This is the video player sub-class for the Ustream embedded video player object.

Event usage: Subscribe to the IBM widget init event that fires on your DOM element to know when the widget has been initialized and you can access its APIs.

Player contructor object for the Ustream video player.
Dependency: videoplayer.js
Used by the video player factory in videoplayer.js file, which is used by the jQuery plug-in.
You can't use this directly. Use the standard jQuery(xxxx).videoplayer() plug-in method to create a video player.

Parameters:

  • [settings] Object optional

    Settings to override defaults and element's @data-xxxx attributes.

                                            jQuery("<your_player_container_selector>").data("widget");
