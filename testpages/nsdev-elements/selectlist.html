Showing a field error

To dynamically show the "field error" on a select widget that's already initialized, do this (replacing your selector as appropriate):

jQuery("#someSelectID").data().select2.$selection.addClass("ibm-field-error");

Official standards & guidelines

ibm.com/standards/web/design/forms/


What

A fancy select list with filtering, a better multi-select option and customized design for v18.


Why

Because sometimes you want the select list to match the them of your design and provide better functionality than the default browser select list element.


Rules & guidelines

  • All select lists in forms are automatically created as a standard select list widget.
  • All standard attributes apply and the HTML for them is standard select HTML. There is no special data-attribute needed.
  • A select list filter will automatically show up if there are at least 8 items in the list.
  • They can be binded with standard jQuery events such as .on("change").

How

Simply create a select list using all standard HTML. Since the select list has a filtering option, the @size attribute has no effect.
