Element classes

  • ibm-textcolor-default
    Sets colored text back to the default text color.
  • ibm-linkcolor-default
    Sets text to the default link color.
  • ibm-textcolor-<color>-<number>
    Colors the text of the element it's applied to unless the element has a specific style that is not meant to be overridden.
  • ibm-alternate-background
    Add to an element to force all text and links to be white. Cannot be overridden by text colors, so only use it when you want everything to be white.

We use the IBM design language color palette verbatim and there are ~140 colors in there. We chose select premium colors to use for text that look good and pass accessibility. Always use these classes and never code your own text color styles. Never.

If you are coloring text and need to set some of it (back) to the default design text color or link color, use these classes:

All text color classes pass accessibility contrast ratios using either white or black backgrounds (as appropriate). Toggling the "accessibility view" will strike through any text color classes that are not accessible with the currently selected background color.

Quick tips:

Anything -60 or higher cannot be used with pure black background.
Anything -40 or lower cannot be used with pure white background.

Official standards & guidelines

ibm.com/standards/web/design/text/


What

Colors, for text.


Why

Because who doesn't love colors?


Rules & guidelines

Deciding what colors to use is up to you. Each color passes accessibility on a white or black background. If you use them on a colored background it is up to you to choose an appropriate text color that passes accessibility for your usage.

  • Text colors can be used on any element anywhere on the page.
  • The ibm-alternate-background class is a convenience class that automatically forces all child text and links white.
  • Color classes cannot be used in combination with the ibm-alternate-background convenience class. They won't work. Use either the all-white convenience class or color classes, but not both.

How

  1. Type some text.
  2. Add a color class to the element.
