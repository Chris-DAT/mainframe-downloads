Official standards & guidelines

ibm.com/standards/web/design/select-list-navigation/


What

A selectlist navigator (formerly known as a "jumpform") is simply a select list that will take the user to the page that they select from the list.


Why

Sometimes you might have a basic long list of links and it might look awkward with your other content layout. So to conserve space you can use the select list navigator.


What's new

In v17 this was known as a "jumpform". In v18 it is now called "selectlist navigator" (makes more sense). The coding is the same as v17 except for the addition of the new standard "data-widget" attribute. This is required to turn the <form> into the widget.


Rules & guidelines

  • Use the below code snippet for a starter.
  • Change the @id to whatever you want. If you have more than one on a page the @id must be a unique value (standard HTML rules).
  • Set the aria-label to a proper title for your select list. This is what a screen reader will read to tell the user what this is.
  • The option values must be fully qualified URLs or else it will not work.

How

  • Copy the sample snippet below
  • Change the @id as appropriate
  • Replace the <option> items with your link URL and texts.
  • If you want to make your select list 100% width of the column it's in, add data-attribute: data-width="100%"
