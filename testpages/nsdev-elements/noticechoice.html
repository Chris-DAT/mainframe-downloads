Official standards & guidelines

ibm.com/standards/web/design/notice-choice/


What

The Privacy Web Service is the method ibm.com web pages must use to display "Notice" and collect "Choice" from individuals when Business Personal Information is collected. The Privacy Web Service manages country translations of "Notice" and "Choices" for opt-in or opt-out for media types and collects the Choices an individual may select from ibm.com web pages.


Why

The Privacy Web Service was developed to encapsulate country specific business rules for all ibm.com supported web pages into one common program. The choices are returned in simple standardized results and e-mail choice is automatically saved to IBM Global Email Cleansing System (GECS).


What's new

In v18, it is the same as v17 except that you do not set your NC options via <meta> tags, you set them in the digitalData object with the rest of the page meta and settings.


Rules & guidelines

You have to have a form and a div placeholder on your page and set your configuration properly according to the Notice and Choice standards wiki. All adoption information and details are available on that wiki site from the N/C team.


How

Add a "nc" child object inside the "pageInfo" object in your digitalData object on your page. Set your config options based on your needs and information from the Notice and Choice standards wiki. The names of the DD properties map and correspond to the same named <meta> tags from v17 and that you see on the wiki. It's not hard to figure out.

			digitalData = {
				page: {
					pageInfo: {
						effectiveDate: "2015-08-03",
						expiryDate: "2022-12-18",
							.... etc ....
						ibm: {
							country: "FR",
							industry: "ZZ",
							.... etc ....
						},
						nc: {
							id : {
								form : "regform",
								email : "emailAddr",
								privacyDiv : "privacyid",
								country : "countryRes"
							},
							questionType : "",
							questionChoice : "",
							pageDesc : "Contact IBM form to send email feedback or questions to IBM in the United States",
							offerCode : "mo1029,mo1039",
							dcSubject : "IM520",
							granular : "off",
							footer : "off",
							preferenceareas : "SVS20GBS,IC957GBS",
							trial : "",
							Submit : "true",
							SaveToGECS : "true"
						}				
					}
				}
			};
