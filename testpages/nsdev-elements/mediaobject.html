Element classes

  • ibm-mo
    Goes on the media object outer container, required to setup a media object element.
  • ibm-mo__image
    Add to the <div> that contains the media object's image.
  • ibm-mo__body
    Add to the <div> that contains the media object's "content".
  • ibm-mo--top
    Image option: Add this class to the media container ibm-mo to top-align the image.
  • ibm-mo--bottom
    Image option: Add this class to the media container ibm-mo to bottom-align the image.
  • ibm-mo--rev
    Image option: Add this class to the media container ibm-mo to reverse the image and place it on the right.
  • ibm-portrait-module-list
    DEPRECATED: This is an old class from v17. It will be removed soon. Use the new media object classes instead.

Official standards & guidelines

ibm.com/standards/web/design/media-objects/


What

The media object is a pattern with an image on one side and content (or other objects) on the other. The image can be on the left or the right.


Why

It's a very basic pattern that has multiple purposes and options. It is often used in threaded commentary like a forum or Facebook.


Rules & guidelines

  • The pattern is an "image with descriptive text" which means an image is required.

How

  • Create a <div class="ibm-mo"> then add two child elements <div class="ibm-mo__image"></div> and <div class="ibm-mo__body"></div> to house your image and your content.
  • Image alignment: The image can be aligned top, bottom, or centered using the modifier classes on the .ibm-mo div. The default is centered, and the optional classs are ibm-mo--top and ibm-mo--bottom. For example, a portrait module uses ibm-mo--top.
  • The image can be placed on the right side by adding the ibm-mo--rev class to the container and you must change the order of the elements and put ibm-mo__image DIV after the ibm-mo__body DIV.
