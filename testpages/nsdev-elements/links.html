Element classes

  • ibm-ind-link
    Used on a <p> to create a single stand-alone individual icon link. Sets proper spacing and style. More than one icon link should use a list with the ibm-link-list class.
  • ibm-link-list
    Used on a <ul> when creating a multiple icon links. Sets proper spacing and style. A single stand-alone individual link should use the <p class="ibm-ind-link"> method because a list with one item is not a list; it is a single item..
  • ibm-icononly
    A modifier used in conjunction with the ibm-ind-link on a stand-alone link <p>. It causes the link text to be hidden and only displays the icon. Allows multiple (and only) icon-only links to be in a row.
  • ibm-inlinelink
    Used in conjunection with an icon class on an inline link. Allows the inline link to have an icon precede it.
  • ibm-icon-after
    Used for special inline icon links (leaving IBM links). Places the icon after the text instead of before it.
  • ibm-icon-nolink
    When added to the text container <p> in conjunction with an icon class, it enables the use of an icon in front of plain text (instead of a link). Used to emphasize things like information or error text. Also used in block links when you want to simulate an icon link by using an icon and the ibm-linkcolor-default class. View the block link element page for examples.
  • ibm-underline
    Force a link to have an underline on static state.
