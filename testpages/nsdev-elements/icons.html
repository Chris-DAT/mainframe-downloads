Icons with links

All examples and variations of how to use icons with links are now on the new links page.


Icons with plain text

Aside from a link, icons can be used with plain text such as an important or informational message.

Your updates were successful. Game on.

Code snippet for above

<p class="ibm-icon-nolink ibm-confirm-link ibm-textcolor-green-50"><span class="ibm-textcolor-default">Your updates were successful. Game on.</span></p>

Icon list

The list of available icon classes are separated into logical lists:

  • Unique icon list: These is the actual single list of unique icons.
  • Social "mono" version: These are forced black/white versions of 3rd party site icons that must be used per their guidelines, with a few extra added.
  • Alias/reuse: These are aliases or alternate use classes that are currently mapped to an existing icon.
  • Deprecated: No longer available as a unique style icon and are mapped to new replacement version icon.

Official standards & guidelines

ibm.com/standards/web/design/icons/


What

Links that have text preceded by an icon, and links that are (visually) the icon only.


Why

A standardized set of icons provides consistency across the enterprise IBM web and provides users with easily recognizable visual cues to types of links.


What's new

Icons are new, some classes are new, some classes have been removed, and there are new types and ways to make icon links. Icons are created as a single font file which allows all icons to be instantly available and cached as a single file, instead of 70 individual files.


Rules & guidelines

  • As with everything else, do not add custom styling to icon links. Use what's available.
  • If there is an icon you need or way to use an icon that you don't see, submit a question or request on the forum or through the v18 standards site.
  • Icons cannot and should not be sized. They are one currently one size, as-is.

How

Use the icons as described below. Icons must be in a standard <p> or <li> element for proper styling.
