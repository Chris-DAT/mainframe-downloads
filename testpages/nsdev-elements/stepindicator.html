Element classes

  • ibm-step-indicator
    Base class used on the widget <div> container to style the step indicator.
  • ibm-tabs
    Properly formats the <ul> as tabs/steps.
  • ibm-container-body
    Tells the steps where the step contents are.
  • ibm-step-container
    Tells the widget this contains a step's content.

Step indicator: Alternate Version

Prior steps are disabled except for the immediate previous step. The user cannot jump to a prior completed step. They must traverse back each step in a linear fashion using the previous button.

The HTML is the exact same as above, just add the data-disableprevious="true" attribute to the widget div.

Official standards & guidelines

ibm.com/standards/web/design/step-indicator/


What

The step indicator allows you to guide the user through a series of steps that they mush complete.


Why

Helps ensure the user completes all necessary steps required for an action.


What's new

Nothing.
