Element classes

  • ibm-hidden-bg-small
    Sets background-image: none on the element when the small grid is being shown.
  • ibm-hidden-bg-medium
    Sets background-image: none on the element when the medium grid is being shown.
  • ibm-hidden-bg-large
    Sets background-image: none on the element when the large grid is being shown.
  • ibm-hidden-bg-xlarge
    Sets background-image: none on the element when the x-large grid is being shown.

The below band has both "ibm-hidden-bg-small" and ibm-hidden-bg-medium". Resize your browser down and you will see the background image disappear for the medium and small grids and the blue background color that was set will show instead. As well, since there is no background image to worry about laying text over, alternate column classes are used for small and medium grids to make the text the full width of the content space (col-12-12).

This is some big text in a band with a background image.

Code snippet for above

<div class="ibm-hidden-bg-small ibm-hidden-bg-medium ibm-padding-top-r2 ibm-padding-bottom-r2" style="background: url(https://ibmdev.webmaster.ibm.com/v18/img/bandimages/snowboarding4.jpg) no-repeat #4178be; background-size:cover">
	<div class="ibm-fluid">
		<div class="ibm-col-12-6 ibm-col-medium-12-12 ibm-col-small-12-12 ibm-center-block ibm-padding-top-1">
			<p class="ibm-h1 ibm-textcolor-white-core">This is some big text in a band with a background image.</p>
		</div>
	</div>
</div>

For information on how to change the column spans based on the grid being used, read the grid override classes information page.

Official standards & guidelines

ibm.com/standards/web/design/hide-images/


What

Allows you to easily hide a background image for a certain grid size(s).


Why

Making background imagery work with text overlaid is tough on every browser width from 1400+ down to 300px for RWD and sometimes you need to just say "this image isn't going to work as a background image on this screen size."


Rules & guidelines

Add the appropriate class to the element that you want to automatically hide the background image for. This will force the background-image to "none". This means that whatever default background color you set for that element will show instead (so make sure you set that if you don't want the default white.)


How

Just add one of these classes to your element (typically used in the lead space or a band.) The classes are similar to the the hide content classes, just with a "-bg" in them:

  • ibm-hidden-bg-xlarge
  • ibm-hidden-bg-large
  • ibm-hidden-bg-medium
  • ibm-hidden-bg-small

For advanced styling, like changing the background image based on the grid size, read the grid classes information page.


Hints and tips

If you want to use a different background image instead of just hiding it, set a style and prefix the selector with the specific grid class.
