Official standards & guidelines

ibm.com/standards/web/design/data-tables/


What

A basic checkbox with a tiny bit of JS that highlights the row in a different color.


Why

For cases when a data table is used for "options" or something in a form and you want the user to be able to "select" the row.


What's new

Nothing.


Rules & guidelines

The checkbox simply colors the background of the row. It's up to you to add any other proper form or JS to do whatever you're doing with the checked items.


How

  • Use proper standard checkbox HTML in the first column of the rows.
  • Add the data-tablerowselector="enable" attribute to your table.

Additional JS/CSS files required

Required supplemental JS and CSS files for this element:

<link href="https://1.www.s81c.com/common../../../css/tables.css" rel="stylesheet">
<script src="https://1.www.s81c.com/common../../../js/tables.js"></script>
