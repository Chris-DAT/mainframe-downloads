Element classes

  • ibm-colored-list
    Add this class along with a standard text color class to the <ul> to enable the bullets to match the color of the text.
  • ibm-plain-list
    Add this class to the <ul> when you have a basic list of links where you want to remove the indent and bullets.
  • ibm-link-list
    Add this class to the <ul> to enable the links to have icons instead of bullets. Use with standard icon link classes.
  • ibm-link-description
    Used in conjunction with a ibm-link-list. When added to the <li>, allows for a matching indented text description under the link.
  • ibm-dropdown-menu
    Add this to a basic <ul> with a list of links to style it as a standard drop-down menu. Sets the width to 100% to fill whatever parent container it is in.
  • ibm-padding-small
    When using ibm-dropdown-menu for a type-ahead/suggestion box, add this class to it to reduce the horizonal padding to match the text input indent.
  • ibm-alpha-list
    Add to a <ol> to make a alphabetical list.
  • ibm-roman-list
    Add to a <ol> to make a roman numeral list.

Official standards & guidelines

ibm.com/standards/web/design/text/


What

When you have a grouping of more than one similar item. A single item is not a list.


Why

Because sometimes a quick bulleted list is easier to scan than reading sentences.
