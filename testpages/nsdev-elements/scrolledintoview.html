Official standards & guidelines

ibm.com/standards/web/design/scrolled-into-view/


What

A utility that will tell you if an element is in the viewport area or not (in the bounds of the viewport).


Why

A helper to use for animations, infinite scroll, and lazy-loading. Because DUH, there's no point in running an animation if it's "offscreen" and the user won't see it.


Rules & guidelines

No rules, it's just a utility you call that returns true or false.


How

Call the utility passing it the element you want to check and it will simply return you true/false:
IBMCore.common.util.scrolledintoview( jQuery("__REPLACE_ME__") );.

There's no magic here, you have to setup your own on-scroll or whatever type event or listener, etc, however you want to use this.
