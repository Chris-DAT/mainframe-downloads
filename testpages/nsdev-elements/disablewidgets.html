All content space widgets can be disabled with a single setting, either by JS or HTML. Anything classified as a "widget" in the JS will be disabled.

To disable via HTML:

Add the data-contentwidgets_enabled="false" attribute to the <body> element.

To disable via JS:

		<script>
			IBMCore.common.util.config.set({
				contentwidgets: {
					enabled: false
				}
			});
		</script>

This page has content space widgets disabled. Notice that the video and carousel HTML below is not converted into widgets. What you see is what is coded in the HTML.

Official standards & guidelines

ibm.com/standards/web/design/disable-widgets/


What

Allows all content space widgets to be disabled and not initialize onload.


Why

For dynamic web apps, SPAs or sites that personalize content and need to initialize widgets based on certain conditions.


What's new

Yes.


Rules & guidelines

This is for advanced sites that need it and know how to initialize widgets manually. This should not be used on standard web pages.
