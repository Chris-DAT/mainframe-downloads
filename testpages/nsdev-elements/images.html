Element classes

  • ibm-resize
    Used on images to set the width to 100% of the container element, allowing them to grow and shrink according to the grid. The height is set automatically to the image's ratio height. A two-way sizing; both grow and shrink.
  • ibm-downsize
    Used on images to set the width to 100% of the container element but will not grow larger than the original size. A one-way sizing; only shrink.
  • ibm-circle
    Adds 50% rounded corners to make a square element (<img>) a circle.

Official standards & guidelines

ibm.com/standards/web/design/imagery/


What

Classes to allow images to scale according to the column they are in.


Why

Because your static image will get cut off when the column shrinks.


Rules & guidelines

When using a small thumbnail like for portrait modules you typically wouldn't want the image to be resized. But for column-width images, like a feature image, you will almost always want to add the "ibm-resize" class to your image.


How

  • Add the class "ibm-resize" to your <img> to have your image sized to 100% width of the container it's in and shrink and grow accordingly.
  • Add the class "ibm-downsize" to your <img> to have your image only sized down from it's original size as the width of the container it's in grows and shrinks. It will not upscale.
