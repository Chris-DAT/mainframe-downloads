Element classes

  • ibm-left
    This left-aligns text. Typically used only when nesting alignments and you need to re-left-align text.
  • ibm-center
    This centers text.
  • ibm-right
    This right-aligns text.
  • ibm-left-small
    This left-aligns text on a small screen that was set with center or right-align class. Useful with ibm-right.
  • ibm-center-block
    This centers a block element using margin:auto. Use on things like tables, columns and other block-level element that are not full-width.
  • ibm-center-position
    This centers an element using left/translate3d. Use on things like stand-alone icons.
  • ibm-vertical-center
    Add this to an element to vertically center the child contents. Uses flex/center. Useful on ibm-fluid.
  • ibm-fleft
    This floats an element left.
  • ibm-fright
    This floats an element right.
  • ibm-inset
    Add this to an image to float it left with proper padding to allow text to wrap around it legibly. Add ibm-alternate to make it a right inset.
  • ibm-mo
    The media object element lets you place an image in a right or left column and content in the other. Use for things like portrait modules, forum posts and threaded commentary.

Centering a block element

To center a block-level element, add the ibm-center-block class to the block element (uses "margin: 0 auto").

v17 v18
ibmweb IBMCore
ibmweb.bitly IBMCore.common.util.bitly

Code snippet for above

		<table class="ibm-data-table ibm-altrows ibm-center-block">
			<thead>
				<tr>
					<th>v17</th>
					<th>v18</th>
				</tr>
			</thead>
			<tbody>
				<tr>
					<td>ibmweb</td>
					<td>IBMCore</td>
				</tr>
				<tr>
					<td>ibmweb.bitly</td>
					<td>IBMCore.common.util.bitly</td>
				</tr>
			</tbody>
		</table>

Vertical centering

By default, column content is top aligned (duh), but there are cases where you want one or two lines of text to be vertically centered with a CTA button. Simply add the ibm-vertical-center class to the ibm-fluid div to vertically center align the columns in the row. Any standard column combination can be used.


This is text with a button in the next column and we don't want to be top aligned with it.

Some link text

Code snippet for above

			<div class="ibm-fluid ibm-vertical-center">
				<div class="ibm-col-12-8">
					<p>This is text with a button in the next column and we don't want to be top aligned with it.</p>
				</div>
				<div class="ibm-col-12-4">
					<p class="ibm-button-link"><a href="##__REPLACE_ME__##" class="ibm-btn-pri">Some link text</a></p>
				</div>
			</div>

Button + icon link alignment

Most often you will have a primary and secondary button together in a row, but sometimes you might want a button and a standard icon link together in a row. To do that, simply make a button in a button row and your icon link next to it, but then add the ibm-fleft class to the button. You may need to add bottom r1 padding if you have an element directly under it, to provide room for the float.

Some important button link A standard icon link

Code snippet for above

			<p class="ibm-btn-row ibm-padding-bottom-r1">
				<a href="##__REPLACE_ME__##" class="ibm-btn-pri ibm-fleft">Some important button link</a>
				<a href="##__REPLACE_ME__##" class="ibm-forward-link">A standard icon link</a>
			</p>

Official standards & guidelines

ibm.com/standards/web/design/text-alignment/


What

Various options for easily aligning elements.


Why

Because everything left-aligned is boring.


Rules & guidelines

Be smart and use alignment wisely. A lot of centered or right-aligned text is not easy to read.


How

  • Use the proper alignment classes for the element type and situation you are trying to align.
  • Like text color classes, alignment classes can be used anywhere on the page.
