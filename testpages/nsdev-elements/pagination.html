Element classes

  • ibm-pagination
    Required base class that goes on the <ul>.
  • ibm-pagination--centered
    Add this class to the UL to center it. To right align, wrap it in a <div class="ibm-right">.
  • ibm-pagination--wide
    Add this to the <ul> to be able to center the pagination page links and float the preview and next links left and right to the edge of the column.
  • ibm-pagination__page
    Add this to a <li> that contains a pagination page link.
  • ibm-pagination__ellipsis
    Add this to a <li> that contains an ellipsis.
  • ibm-pagination__results
    Add to the <li> that contains the "Showing # of #" text.

Official standards & guidelines

ibm.com/standards/web/design/pagination-controls/


What

Pagination is used for such things as displaying a limited number of results on search engine results pages, or showing a limited number of posts when viewing a forum thread. It is used to divide returned data and display it on multiple pages. Pagination also includes the logic of preparing and displaying the links to the various pages.


Why

Pagination is a necessary evil when you have too many items to easily show them all on one screen. Linear content flows such as articles should almost never be broken up into multiple screens. It's better to show the full article on one long screen than to inflict the pain of additional steps on users when all they want to do is read an article, and thus stay within that one item.


What's new

v18 uses updated HTML and instead of <span> separated by pipe | characters, a basic list <ul> is used.


Rules & guidelines

Use of pagination and the various options is dependent on you and your application. There is no hard rule of when, where, or what options to use except some common sense. For example:

  • Page #s can be useful when you have a lot of pages of content, but not required.
  • First/Last page links are ideal when there are a lot of pages, but not required.
  • Previous/Next links can be used with or without labels.
  • You can make pagination links dynamically reload your content or link to a new page, whatever is suitable for your site/app.

How

Making pagination controls is simply a <ul> with classes based on left, right, center align, using the link options you want, and adding the proper accessibility attribute. The example below show all of the variations and options you can do.
