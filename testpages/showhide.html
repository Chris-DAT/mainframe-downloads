Show/hide demo

Overview

There are two toggle states, "show" and "hide." The "show" state displays page links along with their supporting content. The "hide" state displays the links only.

Requirements

Pages that do not adhere to the following requirements may be delayed or disallowed.

  • When implementing the show/hide feature, the "hide" state must be the default selected toggle state.
  • When applied to an entire page, the show/hide feature must appear above the subhead bar at the top of the page, left aligned.
  • Show/hide feature may only be applied to the content space.

Best practices

When developing IBM web pages, you should adhere to the following best practices whenever possible:

  • The label for the show/hide feature must be descriptive (e.g., "Show descriptions"/"Hide descriptions," "Show subcategories"/"Hide subcategories"), or use the appropriate icons to show/hide content.
