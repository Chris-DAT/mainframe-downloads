Host Configuration Assistant for Z Development

ResourcesRequirements

Overview

Host Configuration Assistant for Z Development (HCA) is a web wizard that is designed to simplify planning and configuring of host components for Z Development products. Host Configuration Assistant helps you determine the host components to configure for your products and provides you with feature-based checklists to let you focus on focus. The customized configuration tasks are generated and can be assigned to the programmer to assist the configuration step by step.

User flow

Key values

  • Provides one integrated system for multiple products and host components.
  • Helps determine required and optional components and features dynamically.
  • Provides feature-based checklists that are easy to understand.
  • Enables effective and efficient communication between different roles.
  • Generates customized reports with tasks provided in a recommended order.

Supported products and versions

HCA supports the host configuration for the following products at the latest release levels.

  • IBM Application Delivery Foundation for z/OS, V3.2
  • IBM Wazi for Red Hat CodeReady Workspaces, V1.0
  • IBM Developer for z/OS Enterprise Edition, V14.2
  • IBM Z Open Development, V2.0
  • IBM Developer for z/OS, V14.2
  • IBM Z Open Unit Test, V2.0
  • IBM Debug for z/OS, V14.2
  • IBM Explorer for z/OS, V3.1
  • IBM Fault Analyzer for z/OS, V14.1
  • IBM Dependency Based Build, V1.0
  • IBM File Manager for z/OS, V14.1
  • IBM Z Open Automation Utilities, V1.0
  • IBM Application Performance Analyzer for z/OS, V14.2
  • IBM Wazi Virtual Test Platform, V1.0

Supported host components

  • z/OS Explorer (HALGxxx)
  • z/OS Explorer Extensions (HHOPxxx)
  • z/OS Debugger (HADRxxx)
  • z/OS Source Code Analysis (HAKGxxx)
  • Remote System Explorer API (HAMAxxx)
  • z/OS Dynamic Test Runner (HAL6xxx)
  • Dependency Based Build (HBGZxxx)
  • Z Open Automation Utilities (HAL5xxx)
  • Application Performance Analyzer (HAD0xxx)
  • File Manager (HADLxxx)
  • Fault Analyzer (HADQxxx)
  • COBOL and CICS command Level Conversion Aid (H09Fxxx)

Requirements

  • Supported browsers: Firefox, Chrome, and Safari
  • IBMid
